*Ссылка*
_(от англ. link)_

Алиас, который ссылается (простите за каламбур) на другой алиас. То есть возможность задать связь между уже существующим алиасом и закладкой под другой фразой. Подробное обоснование можно прочитать [здесь](https://t.me/kozaloru/519).

Для создания ссылки используется команда /link. Причём передать можно всю информацию за одну команду: `/link shut up -> money`, где _money_ — алиас уже существующей закладки, а _shut up_ — новый алиас.
